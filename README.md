# README

These file provide the meta-atom layout used for the waveguide simulations in the main figures. Each row represents one meta-atom and contains seven columns:

1. Meta-atom cell index along (x)
2. Meta-atom cell index along (y)
3. Meta-atom (x)-position
4. Meta-atom (y)-position
5. Meta-atom base length
6. Meta-atom base width
7. Meta-atom rotation angle

The data specify the spatial positions, structural dimensions, and rotation-angle distribution of the meta-atoms arranged on the waveguide.
